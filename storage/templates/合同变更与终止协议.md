# 合同变更与终止协议

> **适用场景**：本协议适用于变更或终止已签订的合同场景。
>
> **涵盖情形**：合同条款变更、合同提前终止、合同解除、合同延期等。
>
> **使用前注意**：
> 1. 明确约定变更或终止的具体内容
> 2. 明确约定变更或终止后的处理方式
> 3. 注意对第三方的影响
> 4. 注意已履行部分的处理

## [使用说明]

### 填写指南

1. **{{变量名}}** 格式的内容为必填项，请替换为实际内容
2. **[可选]** 标注的条款根据实际情况保留或删除
3. **[选择A/B/C]** 的内容，请选择合适的选项，删除其他选项
4. **[重要]** 标注的条款请特别注意

### 合同变更与终止协议的关键点：

1. **明确变更或终止类型**：变更 vs 终止 vs 解除
2. **明确费用结算方式**：已支付未履行、已履行未支付的处理
3. **明确交付物处理**：已交付、未交付成果的处理方式
4. **明确知识产权归属**：特别是已交付成果的IP归属
5. **明确原合同效力**：部分失效 vs 全部失效

### 常见错误提示

1. ⚠️ 未明确变更或终止的具体内容 → 可能导致继续按原合同履行
2. ⚠️ 未约定费用结算方式 → 可能导致款项纠纷
3. ⚠️ 未约定交付物处理 → 可能导致知识产权争议
4. ⚠️ 未明确原合同效力 → 可能导致法律关系混乱
