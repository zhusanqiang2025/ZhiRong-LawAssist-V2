# 保密协议（Non-Disclosure Agreement）

> **适用场景**：本协议适用于双方在合作、交易、商务洽谈等过程中需要保护商业秘密的场景。
>
> **使用方式**：
> 1. 作为主合同的附件
> 2. 作为独立协议签署
> 3. 在合作前期先行签署
>
> **使用前注意**：
> 1. 保密期限应根据实际情况合理设定
> 2. 保密信息定义要明确
> 3. 注意保密义务的例外情形

## 第一条 保密信息的定义

### 1.1 保密信息【必填】

本协议所称"保密信息"是指甲方向乙方披露的具有商业价值并标注为"保密"或按商业惯例应视为保密的信息，包括但不限于：

1. **技术信息**：
   - 技术方案、设计图纸、源代码、算法
   - 产品配方、工艺流程、技术参数
   - 研发记录、实验数据、测试结果

2. **经营信息**：
   - 商业计划、营销策略、客户名单
   - 财务数据、成本结构、利润情况
   - 采购渠道、供应商信息、定价策略

3. **管理信息**：
   - 管理层信息、人事资料、组织架构
   - 战略规划、并购计划、投资计划

4. **其他信息**：{{other_confidential_info}}

### 1.2 非保密信息【必填】

以下信息不属于保密信息：
1. 已公开或已成为公知领域的信息（非因乙方泄密所致）
2. 乙方在接收前已合法知悉的信息
3. 乙方从无保密义务的第三方合法获得的信息
4. 法律法规要求披露的信息
5. {{other_exception_info}}

## 第二条 保密义务

### 2.1 保密范围【必填】

乙方应对保密信息承担以下保密义务：
1. 不得向任何第三方披露保密信息
2. 不得将保密信息用于本协议目的以外的用途
3. 采取必要、合理的措施保护保密信息的安全
4. 未经甲方书面同意，不得复制、传播或公开发表保密信息

### 2.2 保密期限【重要】

**[选择A] 固定期限**
乙方对保密信息的保密义务自本协议签署之日起 {{confidential_period}} 年内有效。

**[选择B] 无限期**
乙方对保密信息的保密义务长期有效，直至保密信息成为公知信息。

**[选择C] 分期限**
- 协议履行期间：严格保密
- 协议终止后 {{confidential_period_after}} 年内：继续保密
- 保密信息成为公知信息后：保密义务解除

### 2.3 保密措施

乙方应采取以下保密措施：
1. 建立保密制度，限定接触保密信息的人员范围
2. 对接触保密信息的人员进行保密培训
3. 妥善保管载有保密信息的资料、文件、设备
4. 在保密信息载体上标注"保密"字样

## 第六条 违约责任

### 6.1 违约情形

乙方违反保密义务的情形包括但不限于：
1. 未经授权向第三方披露保密信息
2. 未经授权将保密信息用于其他目的
3. 未采取必要保密措施导致保密信息泄露
4. 本协议终止后未返还或销毁保密信息

### 6.2 违约责任【重要】

乙方违反保密义务的，应承担以下责任：
1. **停止侵害**：立即停止违约行为
2. **赔偿损失**：赔偿甲方因此遭受的所有直接损失和间接损失
3. **违约金**：向甲方支付违约金人民币 {{penalty_amount}} 元
4. **其他责任**：承担其他相应的法律责任

**[风险提示] 违约金数额**：
- 建议根据保密信息的商业价值合理设定
- 可设定为合同总金额的 {{penalty_ratio}}% 或固定金额
- 过高的违约金可能被法院调整

### 6.3 违约金上限

双方确认，本协议约定的违约金不影响甲方根据法律规定要求乙方赔偿实际损失的权利。

## [使用说明]

### 适用场景

1. **商务洽谈前**：在开始商务洽谈前先行签署，保护后续交流的信息
2. **合同附件**：作为主合同的附件，约定保密义务
3. **员工入职**：与员工签署，保护公司商业秘密
4. **投资尽调**：投资方与目标公司签署，保护尽调信息

### 填写指南

1. **保密期限**：
   - 技术秘密：建议选择"无限期"或"长期"
   - 经营信息：可选择"3-5年"
   - 一般商业信息：可选择"1-3年"

2. **违约金**：
   - 应根据保密信息的商业价值设定
   - 技术秘密的违约金可设定较高金额
   - 一般商业信息的违约金可适当降低

3. **保密范围**：
   - 应明确界定保密信息的范围
   - 避免范围过宽导致执行困难
   - 避免范围过窄起不到保护作用

### 常见错误提示

1. ⚠️ 未约定保密期限 → 可能导致保密义务无法确定
2. ⚠️ 未约定违约金 → 可能导致违约成本过低
3. ⚠️ 未约定返还方式 → 可能导致保密信息无法收回
4. ⚠️ 例外情形过宽 → 可能导致保密义务形同虚设
