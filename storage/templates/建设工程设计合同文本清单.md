# 建设工程设计合同文本清单

| 序号 | 文本名称 | 文本要点 |
|------|----------|----------|
| 1 | 建设工程设计合同 |  |
| 2 | 附件1-1：方案设计任务书 | 主要介绍项目概况、设计依据、经济技术指标、市场定位、设计要求、设计工作内容及深度、设计周期。 |
| 3 | 附件1-2：施工图设计任务书 | 本文本分为总则、建筑专业、结构专业、给排水专业、暖通专业、电气专业、屋面、楼地面构造做法、设计配合7个部分，详细列明了各部分要求。 |
| 4 | 附件2：设计团队名单 | 设计团队名单表包含设计师职业资格职称、设计工作年限、主要设计作品信息。文本附主设计师简历。 |
| 5 | 附件3-1：基础数据表1（概念版） | 数据提供给营销、成本、财务使用，并报备置业集团。 |
| 6 | 附件3-2：基础数据表2（方案、施工图版） | 在方案及施工图完成后应分别填写。方案版作为主要技术内容与方案同时上报；施工图版与施工图同时报置业集团备案。 |
| 7 | 附件4：付款通知单 | 设计方完成合同的约定部分工作，达到付款要求，通知发包方在约定时间内支付设计费。 |
| 8 | 附件5：设计成果确认单 | 设计方向发包方确认已完成部分设计工作并已提交全部设计成果。 |
